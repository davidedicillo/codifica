# Starter templates

This folder contains **versioned starter templates** for adopting the Codifica protocol.

## Versions

- `v0.1/`: template files for Codifica protocol v0.1

## How to use

1. Open the version you want (usually `v0.1/`).
2. Copy the files in that folder into the root of your own repository.
3. Commit the files, then start adding tasks to `work.md`.
