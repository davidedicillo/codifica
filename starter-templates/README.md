# Starter templates

## Download / distribution

The public distribution source is the GitHub repository:
`https://github.com/davidedicillo/codifica`

## How to use

1. Open the version you want (usually `v0.2/`).
2. Copy the files in that folder into the root of your own repository.
3. Commit the files, then start adding tasks to `work.md`.
